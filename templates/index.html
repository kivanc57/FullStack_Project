About this project

ALG4 is the abbreviation of ALG which stands for 'Algoritmické zpracování jazyka'in Czech. One can translate it to English as 'Algorithmic language processing'. This website is my project for the zápočet (performance exam) of ALG4 where we have to deploy a website as a final project using:

  • HTML
  • CSS
  • Bootstrap
  • JavaScript
  • Flask on Python
  • Jinja2

Linguistic Data of Text API

This application is aimed to find and illustrate the amount of sentence, the amount of word, the size of the dictionary with interactive graphics.

File Text Search API

This application is aimed to find a text where a word or phrase is given. After the user gives the input, the application finds the containing text within its sentence that possesses the user input.
